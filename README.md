# Cs394_ourChatApp

Project Title: OurChatApp  

Description: OurChatApp is a real-time chat application designed to facilitate communication between users. 
The app includes features such as user authentication, recent chats, user profiles, and a chat interface. 
It leverages Firebase for backend services and Glide for image loading.  


Key Features:  
User Authentication: Allows users to sign up, log in, and manage their profiles.
Real-Time Messaging: Users can send and receive messages instantly.
User Profiles: Each user has a profile that can be viewed and edited.
Recent Chats: Displays a list of recent conversations.
Image Loading: Uses Glide to load and display images efficiently.


Technologies Used:  

Kotlin: The primary programming language for the app.
Java: Used for some parts of the project.
Gradle: Build automation tool.
Firebase: Backend services for authentication and real-time database.
Glide: Image loading and caching library.
Android Studio: IDE for development.

Setup and Run:  
Clone the repository:  
git clone https://github.com/mohammedMoqbel1/ourChatApp_cs394.git
Open the project in Android Studio:  
Launch Android Studio.
Select Open an existing project.
Navigate to the cloned repository and select it.
Sync the project with Gradle files:  
Click on File > Sync Project with Gradle Files.


Run the project:  
Connect an Android device or start an emulator.
Click on the Run button
